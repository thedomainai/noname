## 1. 組織類型

この参照組織は、**AGIネイティブ企業変革会社** とする。

英語では `AI-native Enterprise Design Firm` と呼ぶ。

この会社が売るものは、単なる AI 導入でも、単なる agent platform でもない。  
売るのは次の3つである。

- 企業の将来像の設計
- そこへ至る移行アーキテクチャ
- 実際に入るための wedge module

## 2. 顧客

主顧客は、日本の中堅〜大企業とする。

対象部門:
- CEO / COO 直下の transformation office
- CIO / CDO / CTO
- CHRO / 人事企画
- 経営企画 / 事業企画
- リスク / 法務 / 監査

特徴:
- 規制・コンプライアンス要求が高い
- 部門横断の承認構造が重い
- 既存システムと現場慣行が複雑
- AI導入よりも、 operating model redesign が本質課題になっている

## 3. 提供価値

提供価値は、次の3階層で構成する。

### 3.1 Destination

顧客企業が、AGI/ASI前提でどのような組織・役割・権限・評価・管理会計へ向かうべきかを定義する。

### 3.2 Transition

現在の組織から、その将来像へどう移るかを設計する。  
含むもの:

- role decomposition
- approval redesign
- human-agent handoff
- governance design
- model / tool / data policy
- rollout order

### 3.3 Wedge

顧客に実際に入るための wedge module を提供する。  
例:

- approval redesign kit
- exception handling cockpit
- permissioned context graph
- AI governance trace module

## 4. 参照時点の組織規模

平時の concrete instance として、次の規模を仮定する。

- 人間のフルタイム参加者: 120名
- 契約・提携人材: 40名
- 常時稼働する制度内 AI agent: 1,500
- 顧客アカウント: 35社
- 同時進行の transformation program: 18件
- wedge module の運用案件: 60件

これは巨大企業ではなく、**少数の高密度な human core と多数の agent workforce を持つ高付加価値企業** として設定する。

## 5. 収益モデル

収益は次の4本柱とする。

- strategic design retainer
- transformation program fee
- wedge module subscription
- managed governance / audit service

この構造により、単発コンサルから recurring revenue へ移行しつつ、実装で得た学習を設計資産へ戻せる。

## 6. 非ゴール

この参照組織は、次のものではない。

- 汎用AI戦略コンサル
- 汎用社内チャット / 汎用 copilot 企業
- 単なる受託開発会社
- 基盤モデル会社
- 単なる agent runtime 企業

## 7. リスクプロファイル

この組織が主要に背負うリスクは次の通り。

- 顧客企業の制度設計を誤ることによる大きな downstream impact
- 顧客データ・従業員データ・経営判断の高機密性
- AI提案の rubber-stamp 化
- 顧客内の政治的対立への巻き込まれ
- wedge が将来の operating model に資産蓄積しないこと
- 自社内部で人間が review clerk に劣化すること

## 8. なぜこの参照組織を使うか

この組織類型は、`AI-native institution / OS` の論点を最もよく露出させる。

理由:
- 目的、権利、正当性、分配、文化、人間発達、情報入口、可観測性がすべて重要
- 単なる SaaS 企業よりも制度設計の密度が高い
- 顧客への提案と自社運営の両方で OS 設計を要求される
- `destination / transition / wedge` の3層を持ち、 abstract spec を具体に落としやすい

したがって、この profile は `AI-native institution / OS` の最初の concrete instance として妥当である。
