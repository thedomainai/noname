因果連鎖CAUSAL CHAINS
4 つの因果連鎖 4 Causal Chains
次元は独立ではなく、4 つの因果チェーンで接続される。いずれかの次元が欠落すると、チェーン全体が機能しない。 Dimensions are not independent—they are connected through 4 causal chains. If any dimension is missing, the entire chain fails.
CHAIN A
Objective → Decision → Actuation → Incentives
目的境界 → 決定権限 → 実行面 → インセンティブ設計。OS が現実を一貫して動かすための最短経路。 Objective → Decision rights → Actuation → Incentive design. The shortest path for the OS to consistently drive reality.
CHAIN B
Membership → Decision → Operating → Human Dev → Legitimacy
権利付与 → 権限設計 → 日常の運用 → 人間の成長 → 正当性維持。人間参加が象徴に堕さないための連鎖。 Rights → Authority design → Daily operations → Human growth → Legitimacy. The chain that prevents human participation from becoming symbolic.
CHAIN C
World Model → Deliberation → Decision → Actuation → Safety → Meta-Gov
正規状態 → 推論 → 承認 → 実行 → 観測 → OS 更新。制度知能が成立するための認知ループ。 World model → Reasoning → Approval → Execution → Observation → OS update. The cognitive loop for institutional intelligence.
CHAIN D
Legitimacy → Objective → Political Economy → Social Order → Legitimacy
正当性 → 目的 → 分配 → 社会秩序 → 正当性。正当性は入力でも出力でもある循環構造。 Legitimacy → Objective → Distribution → Social order → Legitimacy. A feedback loop where legitimacy is both input and output.
次元のチェーン参加回数 Dimension Chain Participation
D4（Decision Rights）が 3 チェーンに参加し、制度全体のハブとして機能する。 D4 (Decision Rights) participates in 3 chains, serving as the hub of the entire institution.
